# Sister Site Content Drafts — sub-zerorepairservices.com

This folder contains paste-ready content for **sub-zerorepairservices.com**, the sister site to fivestarappliancerepairpros.com. The goal is to differentiate the sister site enough to avoid Google's scaled-content classifier while keeping it pursuing the same high-value keyword universe.

## Why this exists

Audit confirmed the two sites currently share:
- Same phone number (800) 651-4528
- Verbatim H1 ("Same-Day Sub-Zero Appliance Repair Across South Florida")
- Verbatim hero subhead
- Identical 5-service catalog
- Identical 7-county service area
- Same 6-section homepage structure

Google sees this as duplicate content, which puts both domains at risk of suppression. This rewrite breaks the duplication signals while preserving the keyword targeting.

## Strategy

Five Star already ranks **#1 for all 10 primary "Sub-Zero ___ Miami FL" keywords** (per the May 2026 SERP report). The sister site won't compete head-on for those right away — that decision is on hold pending owner discussion (Layer 1 in the keyword plan).

Instead, the sister site targets:

- **Tier 2** — adjacent geo modifiers Five Star isn't dominating yet (Fort Lauderdale, Boca Raton, Coral Gables, Aventura, Sunny Isles, etc.)
- **Tier 3** — symptom + commercial intent ("Sub-Zero refrigerator not cooling Miami", "Sub-Zero error code E1", same-day / 24/7 / emergency variations)
- **Tier 4** — marine specialty (yachts, dockside service across SoFL marinas)
- **Tier 5** — brand co-mentions in body copy (Wolf, Cove, Viking, Thermador, Bosch, Miele) — no dedicated brand pages
- **Tier 6** — long-tail / model-specific (Pro 48, 648, 690, 532, Designer, Classic, BI-36, BI-42, Column, Integrated)

## Differentiation Tactics

| Element | Five Star | Sister Site |
|---|---|---|
| **Phone** | (800) 651-4528 | **MUST be different forwarding number** (e.g., CallRail / Twilio) |
| **Brand voice** | "Established expert, since 1994" | "Specialist-led, rapid-response, technician-first" |
| **Tone** | Informational paragraphs | Bullet-led + scannable + first-person from technician |
| **Author** | Marvin Schuster | **Daniel Rivera, Senior Service Technician** |
| **URL pattern** | `/services/refrigerator-repair/` | `/sub-zero-refrigerator-repair-miami-fl/` (keyword-rich flat URL) |
| **City URL** | `/areas-we-service/miami-dade/coral-gables/` | `/sub-zero-repair-coral-gables-fl/` |
| **Hero structure** | Image right + text left | Stat-led intro + technician quote + CTA stack |
| **FAQ position** | End of page | Mid-page (right after direct answer) |
| **Sources cited** | Sub-Zero use & care guides | Sub-Zero error codes + service bulletins (different sub-pages of same domain) |
