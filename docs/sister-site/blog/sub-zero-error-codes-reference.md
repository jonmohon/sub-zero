### Lead (40-60 word direct answer for GEO)

Modern Sub-Zero refrigerators self-diagnose and display error codes when something fails. The most common codes are E1 (refrigerator over-temperature), E2 (freezer over-temperature), E3 (sealed-system pressure), EE (electronic control fault), and dr (door open too long). Here's what each one actually means, what to check first, and when it's time to call for service.

---

### How Sub-Zero Error Codes Work

Sub-Zero electronic control boards continuously monitor temperatures, sensor readings, and component status. When a reading falls outside the expected range, the unit displays an error code on the digital readout.

Two important things to know:

1. **An error code doesn't necessarily mean the unit needs service.** Some errors clear themselves once the underlying condition resolves (a long door-open event, for example).

2. **An error code is a starting point, not a diagnosis.** "E1" tells us the refrigerator is too warm. It doesn't tell us why. The why is what we figure out on-site.

That said, the error code is your most valuable troubleshooting clue. Knowing the code before you call means we arrive with the right parts and the service bulletin already pulled.

## The Codes

### E1 — Refrigerator Section Over-Temperature

**What it means:** The refrigerator compartment is reading higher than its setpoint. Usually the refrigerator section has been at or above 55°F for an extended period.

**What to check first:**
- Was the door left open recently?
- Is the unit packed too tightly (blocking airflow)?
- Did you put a large amount of warm food in?

If any of these — wait 4-6 hours. The code may clear on its own as the unit recovers.

**If E1 persists:** the most common causes are evaporator fan failure (cold isn't being blown into the cabinet), defrost system failure (ice buildup blocking airflow), or compressor/sealed-system issue. Service call.

---

### E2 — Freezer Section Over-Temperature

**What it means:** The freezer compartment is reading higher than its setpoint, usually above 15°F for an extended period.

**What to check first:**
- Was the freezer door left open?
- Is the freezer drawer not sealing fully?
- Did you load a large amount of unfrozen food?

**If E2 persists:** evaporator fan failure (most common), defrost system failure, or freezer-side compressor issue on dual-compressor units (Pro 48). Service call.

---

### E3 — Sealed-System Pressure Issue

**What it means:** The sealed refrigeration system is reading abnormal pressure. This is a serious code — it indicates the refrigerant circuit itself has a problem.

**What to check first:**
- Has the back of the unit been against a wall (heat trap)? Pull it forward 2 inches and let it run for 24 hours.
- Has there been an unusual condenser blockage?

**If E3 persists:** sealed system service is required. Could be refrigerant loss, restriction in the capillary tube, condenser failure, or compressor weakness. Service call — this isn't a DIY fix and the longer you run with E3 active, the more risk to other components.

---

### EE — Electronic Control Board Fault

**What it means:** The main control board has detected an internal fault. Could be a sensor reading errors, communication failure between boards, or a board-internal issue.

**What to check first:**
- Power-cycle the unit. Unplug for 5 minutes, plug back in. EE codes from transient electrical issues sometimes clear on a power cycle.
- Check that all visible cables behind the kick plate are seated.

**If EE persists:** electronic control board replacement or sensor replacement. Service call. We carry OEM control boards for the BI series, Designer columns, and most Pro 48 configurations on every truck.

---

### dr — Door Alarm

**What it means:** A door has been open too long (typically more than 5 minutes).

**What to check first:**
- Look — is a door actually open or ajar?
- Check that the door is fully closed and the gasket is sealing.
- Verify nothing is obstructing the door (a tall bottle pushed against the back, for example).

**If dr persists with the door clearly closed:** the door switch is failing. The unit thinks the door is open. Easy fix, common part. Service call.

---

### F1 / F2 / F3 — Fan-Specific Errors

**What it means:** The unit has detected a fan motor failure. F1 typically refers to evaporator fan, F2 to condenser fan, F3 to a secondary fan on dual-evaporator units.

**What to check first:**
- Is there obvious ice buildup in the freezer (which would jam the evaporator fan)?

**If the F-code persists:** fan motor replacement. Common failure, common part. Service call. Most fan motor replacements take under an hour.

### Pro 48 / Pro 36 Specific Codes

The Pro series uses an extended code set:

| Code | Meaning |
|---|---|
| **A1** | Refrigerator over-temperature warning |
| **A2** | Freezer over-temperature warning |
| **A3** | Door alarm |
| **A4** | High-temperature alarm (extended over-temp event) |
| **C1** | Refrigerator compressor fault |
| **C2** | Freezer compressor fault |
| **C3** | Condensor fan fault |
| **F1** | Refrigerator evaporator fan fault |
| **F2** | Freezer evaporator fan fault |
| **TS** | Temperature sensor fault |

Most Pro 48 codes resolve only with a service call — the dual-compressor architecture is more complex than the single-compressor BI series.

---

### Designer Series Codes

Designer columns use a slightly different code set. The Designer column uses an icon-based fault display rather than alphanumeric codes — typically a flashing snowflake or thermometer icon indicating a temperature-out-of-range condition. The associated control board can be queried for the specific fault — that's a service-level diagnostic.

## How to Reset a Sub-Zero Error Code

Most Sub-Zero error codes will clear automatically once the underlying condition resolves. If you want to manually reset:

1. **Power-cycle the unit.** Unplug at the wall (or trip the breaker if hardwired) for 5 minutes. Plug back in.
2. **Check the display.** If the error code returns within 24 hours, the underlying issue has not resolved. Call for service.

**Important:** never disable an error code by repeated power-cycling without addressing the cause. Sub-Zero error codes exist to protect the unit. Ignoring an E3 (sealed system pressure) code, for example, can lead to compressor damage worth thousands of dollars to replace.

## When to Call for Service

Call us if:

- The error code persists after 24 hours
- The error code is E3 (sealed system) or EE (control board) — these don't self-resolve and waiting risks further damage
- You see multiple error codes simultaneously
- The unit's cooling is materially affected even if no error code is showing

When you call, have these ready:
- Model number (metal plate inside the refrigerator section)
- The exact error code you're seeing
- How long it's been showing
- Any troubleshooting you've already tried

We'll arrive with the OEM parts most likely needed and the service bulletin for your specific code already pulled.
